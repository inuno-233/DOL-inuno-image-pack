# DOL-inuno-image-pack
基于DOL原版游戏上修改了部分代码制作的美化模组
<br>A visual enhancement mod created by modifying portions of the original DOL game's code.
<br><b>关于权限</b>：允许其他创作者对该模型的内容进行修改，允许创作新的替换图片、新增衣物等，当创作者绘画了相应服饰替换可与创作者本人联系整合进包体；注意：创作中<b>禁止出现R18G向作品，例如鲜血、残肢等作品</b>，禁止使用含有版权争议或明确版权归属的绘画作品作为参考进行绘画移植，例如everskies。
